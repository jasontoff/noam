# noam

Projects for Noam.

## fps-game

Multiplayer FPS arena game built with Three.js and Socket.IO.
Live at [noam.bot/fps-game](https://noam.bot/fps-game/).

### Run locally

```
cd fps-game
npm install
npm start
```

Then open http://localhost:3000

## racing-game

Arcade racer with single-player and multiplayer (vs NPCs) modes.
Built with Three.js.

### Run locally

```
cd racing-game
npm install
npm start
```

Then open http://localhost:3001

Controls: WASD or arrow keys to drive, SPACE for handbrake.
